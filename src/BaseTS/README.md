## BaseTS
> Base Template written in TypeScript


## Usage

> First, cd to [bin](https://github.com/tingwei628/react-component/tree/master/src/BaseTS/bin)

- build
```
$make build
```

- minify js, css, public resources, and etc.
> Uglify-es
```
#make minify
```

- run
```
$make run
```

- test
```
$make run
```

- lint
> tslint
```
$make lint
```
